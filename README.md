無料タスク管理ツール

概要
シンプルで使いやすい、個人向けのタスク管理ツールです。
タスクの整理、期限管理、タグ付けなどが直感的に行えます。

主な機能
・タスク管理: 作成、編集、完了、削除
・期限設定: カレンダーによる日付指定
・詳細メモ: タスクごとのメモ記述
・リスト整理: プロジェクト（フォルダ）分け
・タグ付け: ラベルによる分類（ドラッグ＆ドロップ対応）
・検索: タイトルやメモのキーワード検索
・並び替え: 作成日や期限日でのソート

開発者向け情報
・構成: HTML + JavaScript (ES Modules) + Tailwind CSS
・バックエンド: Firebase (Auth, Firestore)
・ホスティング: Cloudflare Workers (静的配信)
・その他：Git,GitHub,VS Code,Gemini,Grok
